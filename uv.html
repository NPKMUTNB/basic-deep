คู่มือใช้งาน uv เบื้องต้น (Quickstart)

สรุปขั้นตอนติดตั้งและใช้งานพื้นฐานของ uv อิงจากเอกสารทางการ docs.astral.sh/uv (โปรดดูเอกสารทางการสำหรับรายละเอียดครบถ้วน)

uv คือเครื่องมือจัดการแพ็กเกจและโปรเจกต์ Python ที่รวดเร็วมาก รวมความสามารถของ pip/pip-tools/pipx/poetry/pyenv/virtualenv ไว้ในเครื่องมือเดียว

สร้างและใช้งาน venv

uv venv                # สร้าง virtual environment (โฟลเดอร์ .venv)
  

    
ติดตั้ง uv

    
macOS/Linux:

    
curl -LsSf https://astral.sh/uv/install.sh | sh

    
หลังติดตั้ง แนะนำให้อ่านหน้า “First steps” และ “Installation” บนเว็บไซต์ทางการ

  


  

    
คำสั่งพื้นฐาน

    
    
      
  • uv init: สร้างโปรเจกต์ใหม่
    • 
      
  • uv add / uv remove: เพิ่ม/ลบ dependencies
    • 
      
  • uv run: รันคำสั่งหรือสคริปต์ในสภาพแวดล้อมของโปรเจกต์
    • 
      
  • uv lock / uv sync: อัปเดต lockfile และซิงก์ environment
    • 
      
  • uv venv: สร้าง virtual environment
    • 
    
    เมื่อใช้ uv run / uv pip ไม่จำเป็นต้อง activate venv
    
  

      
  • uv tool / uvx: รันและติดตั้ง CLI tools
    • 
      
  • uv python: จัดการเวอร์ชัน Python
    • 
      
  • uv pip: อินเทอร์เฟซที่เข้ากันได้กับ pip/pip-tools
    •

    เริ่มโปรเจกต์ใหม่

    uv init myapp
cd myapp
uv add ruff
uv run ruff --version
uv lock
uv sync
    uv จะสร้าง/ใช้ .venv ให้โดยอัตโนมัติเมื่อใช้ uv run / uv sync

    สคริปต์เดี่ยว (PEP 723)

    echo 'import requests; print("ok", bool(requests))' > script.py
uv add --script script.py requests
uv run script.py

    เครื่องมือ (Tools)

    uvx pycowsay 'hello world'
uv tool install ruff
ruff --version

    จัดการ Python เวอร์ชัน

    uv python install 3.10 3.11 3.12
uv venv --python 3.12.0
uv python pin 3.11

    อินเทอร์เฟซสไตล์ pip

    uv pip compile requirements.in --universal -o requirements.txt
uv venv
uv pip sync requirements.txt
uv pip install requests
uv pip uninstall requests
